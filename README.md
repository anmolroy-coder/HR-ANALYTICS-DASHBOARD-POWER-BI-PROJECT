## HR Analytics Dashboard (Power BI)
## Project Overview
• This project is an interactive HR Analytics Dashboard built using Power BI to analyze employee data and gain meaningful insights related to attrition, salary, age, and job roles.

• The goal of this project is to transform raw HR data into a visual and interactive dashboard that helps in better decision-making.

## KPIs (Key Performance Indicators)
• Total Employees: 1470

• Attrition Count: 237

• Attrition Rate: 16.1%

• Average Age: 37

• Average Salary: 6.5K

• Average Working Years: 7.0

## Data Processing Steps
• Imported raw HR dataset (CSV)

• Data cleaning using Power Query

• Removed duplicates

• Handled missing/null values

• Changed data types

• Created calculated columns (Attrition Count)

• Created measures (Attrition Rate)

## Dashboard Features
• Interactive and dynamic visuals

• Department-wise filtering using Slicer (HR, R&D, Sales)

• Multiple charts for deep insights

## Visualizations Used
• Donut Chart → Attrition by Education

• Column Chart → Attrition by Age

• Matrix Table → Job Role Analysis

• Bar Chart → Attrition by Salary Slab

• Area Chart → Attrition by Years at Company

• Stacked Bar Chart → Attrition by Job Role

## Insights Gained
• Higher attrition observed in certain job roles

• Employees with lower salary slabs have higher attrition

• Most attrition occurs in early years at the company

• Age group 26–35 shows highest attrition

## Tools & Technologies
• Power BI

• Power Query

• DAX (Data Analysis Expressions)
